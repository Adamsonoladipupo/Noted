Context-Aware Note Taking App
This is a note-taking application designed to help users save links along with personal context. 
Instead of just bookmarking content, users can capture *why* a link matters, making it easier to recall ideas, insights, and intentions later.

     “Remember why you saved something, not just what you saved.”

🚀 Features
🔗 Smart Link Saving
* Save links with automatic metadata:
  * Title
  * Description
* Supports articles, videos, tweets, and more

✍️ Personal Snippets
* Add a short note explaining:
  * Why the link is important
  * Key takeaways
  * Actions to revisit later

🏷️ Tagging & Organization
* Organize notes using custom tags
* Group items into collections or folders

🔍 Search
* Search by:
  * Keywords
  * Tags
  * Link titles/descriptions

⏰ Reminders (Optional)

* Set reminders to revisit saved content
* Rediscover older notes over time

📝 Offline Notes

* Create notes without links
* Store ideas, thoughts, and quick drafts
